# Dear Bbu ❤️ - Romantic Apology Website

A heartfelt and interactive romantic apology website built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Features

✨ **Beautiful UI** - Modern design with gradient backgrounds and glowing text effects

🎵 **Background Music** - Toggle music on/off with a smooth floating button

💌 **Heartfelt Messages** - Multiple letters expressing genuine apology

✨ **Animated Floating Emojis** - Decorative animated hearts, roses, and sparkles

📱 **Fully Responsive** - Looks great on all devices (mobile, tablet, desktop)

🎨 **Smooth Animations** - Powered by Framer Motion for delightful interactions

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Vite** - Build tool

## Getting Started

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The app will open at `http://localhost:3000`

## Deployment Options

### Option 1: GitHub Pages (Free)
```bash
npm run build
```
Then enable GitHub Pages in repository settings, pointing to the `dist` folder.

### Option 2: Vercel (Recommended - Free)
1. Push to GitHub
2. Go to vercel.com
3. Import repository
4. Deploy automatically!

### Option 3: Netlify (Free)
1. Push to GitHub
2. Go to netlify.com
3. New site from Git
4. Build command: `npm run build`
5. Publish directory: `dist`

### Option 4: Replit
1. Fork on Replit
2. Add `.replit` file with: `run = "npm install && npm run dev"`
3. Click Run

## Customization

### Change the Messages
Edit the `letters` array in `src/App.tsx`

### Change Colors
Modify Tailwind classes or `tailwind.config.js`

### Change Music
Replace the audio URL in `src/App.tsx`
